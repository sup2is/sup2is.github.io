# multi-thread와 race-condition

multi-thread 환경에서 공유되는 자원을 서로 다른 thread가 동시에 접근해서 변경하려고하면 race-condition이 발생한다. 이런 race-condition은 데이터 정합성과 크게 연관되어 있기 때문에 multi-thread환경에서는 반드시 주의해야 한다.
